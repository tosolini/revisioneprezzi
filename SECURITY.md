# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | :white_check_mark: |

## Reporting a Vulnerability

To report a security vulnerability, please open a private security advisory at:

https://github.com/tosolini/revisioneprezzi/security/advisories/new

You will receive a response within 48 hours. If the issue is confirmed, a fix will be
released as soon as possible depending on complexity.
